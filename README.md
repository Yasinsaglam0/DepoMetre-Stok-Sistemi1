🍦 DepoMetre - Akıllı Envanter Yönetim Sistemi

DepoMetre, özellikle dondurma dükkanları gibi hızlı devir yapan işletmeler için geliştirilmiş, stok takibini dijitalleştiren ve karar destek mekanizması sunan bir masaüstü yazılımıdır.

🚀 Proje Amacı
Bu proje, manuel stok takibi sırasında oluşabilecek insan hatalarını en aza indirmek ve işletme sahibine hangi ürünün ne zaman sipariş edilmesi gerektiğini görsel uyarılar (Kritik Stok Sistemi) ile bildirmek amacıyla tasarlanmıştır.

✨ Temel Özellikler
* **Dinamik Stok Takibi:** Ürünlerin kategori, miktar ve birim bazlı yönetimi.
* **Kritik Eşik Uyarı Sistemi:** Stok seviyesi belirlenen limitin altına düşen ürünlerin otomatik olarak **Kırmızı** renk ile vurgulanması.
* **Canlı İstatistik Paneli:** Toplam çeşit sayısı, acil sipariş bekleyen ürün sayısı ve birim bazlı toplam envanter dağılımının (Kg/Adet) anlık hesaplanması.
* **Hızlı Arama:** LIKE sorgusu ile ürünler arasında saniyeler içinde filtreleme.
* **CRUD İşlemleri:** Tam kapsamlı ürün ekleme, silme ve güncelleme desteği.

 🛠 Teknik Detaylar
* **Programlama Dili:** C# (.NET Framework)
* **Veritabanı:** MS SQL Server (LocalDB)
* **Geliştirme Ortamı:** Visual Studio 2022
* **Mimari:** Katmanlı Mimari ve Veritabanı Entegrasyonu

📖 Kurulum ve Kullanım
1. Projeyi GitHub üzerinden `Clone` edin veya `ZIP` olarak indirin.
2. Visual Studio ile `.sln` dosyasını açın.
3. Veritabanı yolu (Connection String) `AttachDbFilename` olarak ayarlandığı için ek bir SQL kurulumu gerekmeden direkt çalıştırabilirsiniz.
